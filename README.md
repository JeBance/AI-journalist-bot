# AI-journalist Bot

Бот для автоматической публикации постов из AI-journalist инфраструктуры в Telegram.

---

## 🚀 Быстрая настройка

### Шаг 1: Создать бота

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Введите имя бота (например, `AI Journalist Bot`)
4. Введите username бота (например, `ai_journalist_bot`)
5. **Сохраните API Token** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Добавить бота в администраторы канала

1. Откройте ваш канал
2. Зайдите в **Управление каналом** → **Администраторы**
3. Нажмите **Добавить администратора**
4. Найдите вашего бота по username
5. Дайте права на **публикацию сообщений**
6. Сохраните

### Шаг 3: Получить Chat ID канала

**Способ 1: Через бота**

1. Отправьте любое сообщение в канал
2. Перешлите это сообщение боту [@GetMyIDBot](https://t.me/GetMyIDBot)
3. Бот покажет `Chat ID` (обычно начинается с `-100`)

**Способ 2: Через API**

```bash
curl "https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates"
```

В ответе найдите `"chat":{"id":-100XXXXXXXXXX`

### Шаг 4: Настроить конфигурацию

1. Скопируйте пример конфигурации:

```bash
cd /root/git/AI-journalist-bot
cp config.json.example config.json
```

2. Отредактируйте `config.json`:

```json
{
  "token": "123456789:ABCdefGHIjklMNOpqrsTUVwxyz",
  "channel_id": "-1001234567890"
}
```

### Шаг 5: Проверить работу

```bash
python3 publisher.py test
```

Если всё настроено правильно, увидите:
```
✅ Бот найден: @ai_journalist_bot (AI Journalist Bot)
```

## 📖 Использование

### Публикация из командной строки

```bash
python3 publisher.py "Текст поста в Markdown V2"
```

**Пример:**

```bash
python3 publisher.py "🔥 *PHP 8.4.0 released*

Вышел финальный релиз PHP 8.4\.

#php #javascript"
```

### Публикация из Python

```python
from publisher import publish

result = publish("Текст поста")

if result["success"]:
    print(f"Опубликовано! Message ID: {result['message_id']}")
else:
    print(f"Ошибка: {result['error']}")
```

### Интеграция с AI-агентом

AI-агент может использовать бота после загрузки папки:

```python
# AI-агент загружает эту папку и использует:
import sys
sys.path.insert(0, '/root/git/AI-journalist-bot')
from publisher import publish

# После написания поста:
publish(текст_поста)
```

## ⚠️ Форматирование

Бот поддерживает **Telegram Markdown V2**.

### Экранирование символов

В Markdown V2 нужно экранировать эти символы:

```
_ * [ ] ( ) ~ ` > # + - = | { } . !
```

**Пример:**

```python
# ❌ Неправильно:
publish("PHP 8.4 > PHP 8.3")

# ✅ Правильно:
publish("PHP 8\.4 \> PHP 8\.3")
```

### Поддерживаемое форматирование

| Элемент | Синтаксис |
|---------|-----------|
| **Жирный** | `*текст*` |
| _Курсив_ | `_текст_` |
| `Моноширинный` | `` `текст` `` |
| [Ссылка](url) | `[текст](url)` |
| ```код``` | ```` ```код``` ```` |

## 🐛 Решение проблем

### Ошибка: "Chat not found"

**Причина:** Бот не добавлен в администраторы канала

**Решение:**
1. Проверьте, что бот добавлен в канал
2. Проверьте, что у бота есть права на публикацию
3. Проверьте Chat ID (должен начинаться с `-100`)

### Ошибка: "Unauthorized"

**Причина:** Неверный токен бота

**Решение:**
1. Проверьте токен в `config.json`
2. При необходимости получите новый токен у @BotFather

### Ошибка: "Bad Request: can't parse entities"

**Причина:** Ошибка в Markdown V2

**Решение:**
1. Проверьте экранирование спецсимволов
2. Убедитесь, что все теги закрыты
3. Используйте `parse_mode=None` для отключения форматирования
